# 楼宇资产管理平台功能需求

## 1. 楼宇资产管理

### 1.1 资产数据管理

#### 功能描述
- 根据用户权限，可按全区或按街道板块维护资产数据
- 支持资产信息的新增、查询、查看、变更、删除及数据统计
- 提供批量导入导出功能

#### 数据字段
| 字段名称 | 字段类型 | 是否必填 | 说明 |
|---------|---------|---------|------|
| 资产编号 | 文本 | 是 | 系统自动生成，唯一标识 |
| 资产名称 | 文本 | 是 | 资产的名称 |
| 所属街道 | 下拉选择 | 是 | 资产所属的街道 |
| 载体地址 | 文本 | 是 | 详细地址信息 |
| 经度 | 数值 | 否 | 地理位置经度坐标 |
| 纬度 | 数值 | 否 | 地理位置纬度坐标 |
| 土地性质 | 下拉选择 | 否 | 如：商业用地、工业用地等 |
| 建筑总面积 | 数值 | 是 | 单位：平方米 |
| 可出租面积 | 数值 | 否 | 单位：平方米 |
| 资产标签 | 多选标签 | 否 | 资产特征标签 |
| 资产图片 | 图片 | 否 | 支持多张图片上传 |
| 资产描述 | 长文本 | 否 | 资产详细描述 |
| 使用日期 | 日期 | 否 | 资产启用日期 |
| 状态 | 下拉选择 | 是 | 正常/停用 |

#### 统计功能
- 按全区/街道统计资产数量
- 按全区/街道统计建筑总面积
- 按全区/街道统计可出租面积
- 资产分布统计（按街道、按类型）

### 1.2 楼宇信息维护

#### 1.2.1 园区楼宇管理
管理园区内的楼宇信息，支持多栋楼宇的关联管理。

**数据字段**：
- 所属园区资产（关联资产）
- 楼宇编号（系统生成）
- 楼宇名称
- 楼宇简称
- 载体投资主体
- 坐落位置
- 建筑面积（平方米）
- 可出租面积（平方米）
- 建筑高度（米）
- 楼层数量
- 楼宇标签
- 楼宇图片（支持多张）
- 楼宇描述
- 建成日期
- 状态（正常/停用/装修）

#### 1.2.2 独栋楼宇管理
管理独立的楼宇信息，不隶属于任何园区。

**数据字段**：
- 与园区楼宇相同的基础字段
- 直接关联到资产，无需选择园区

### 1.3 楼层管理

#### 功能描述
- 维护楼宇下每一层楼的信息
- 支持楼层信息的增删改查
- 支持楼层平面图上传

#### 数据字段
| 字段名称 | 字段类型 | 是否必填 | 说明 |
|---------|---------|---------|------|
| 所属楼宇 | 关联选择 | 是 | 关联楼宇信息 |
| 楼层编号 | 文本 | 是 | 系统自动生成 |
| 楼层名称 | 文本 | 是 | 如：1F、2F、B1等 |
| 楼层号 | 数值 | 是 | 楼层的数字标识 |
| 建筑面积 | 数值 | 是 | 单位：平方米 |
| 可出租面积 | 数值 | 否 | 单位：平方米 |
| 楼层类型 | 下拉选择 | 否 | 办公/商业/仓储等 |
| 楼层平面图 | 文件 | 否 | 支持图片或CAD文件 |
| 备注 | 长文本 | 否 | 其他说明信息 |
| 状态 | 下拉选择 | 是 | 正常/装修/停用 |

### 1.4 房间管理

#### 功能描述
- 管理楼层内的房间资源
- 支持房间状态的实时更新
- 提供房间资源的查询统计

#### 数据字段
- 所属楼层（关联选择）
- 房间编号（系统生成）
- 房间号（如：101、102）
- 房间面积（平方米）
- 房间类型（办公室/会议室/仓库等）
- 房间状态（空置/已租/装修/自用）
- 区域位置（东/南/西/北/中）
- 资产权属
- 备注信息

## 2. 资产地图

### 2.1 地图展示

#### 功能描述
- 集成天地图政务版地图引擎
- 支持电子矢量地图和倾斜摄影两种底图切换
- 提供地图基础操作（放大、缩小、移动、测距等）

#### 显示模式
1. **全区视角**：默认显示全区范围，展示所有资产分布
2. **街道视角**：支持下钻到具体街道，查看街道内资产分布
3. **资产详情**：点击资产图标，弹出资产概要信息

### 2.2 资产分布展示

#### 地图标注
- 不同类型资产使用不同图标
- 资产状态通过颜色区分（正常/停用）
- 支持按条件筛选显示

#### 资产概要信息
点击地图上的资产图标，显示：
- 资产名称
- 资产状态
- 资产标签
- 缩略图
- 建筑面积
- 下属楼宇数量
- 查看详情按钮

### 2.3 地图工具

- 图层控制（底图切换）
- 测量工具（距离、面积）
- 定位工具（快速定位到指定位置）
- 搜索工具（按名称搜索资产）

## 3. 楼宇载体数据

### 3.1 数据统计维度

#### 3.1.1 全区统计
展示全区楼宇载体的整体情况：

**空间资源统计**：
- 楼宇总数
- 总建筑面积
- 可租售面积
- 已出租面积
- 空置面积
- 空置率

**载体分类统计**：
- 政府载体数量及面积
- 社会载体数量及面积
- 各类型载体占比分析

#### 3.1.2 街道统计
按街道维度展示楼宇载体情况：
- 各街道楼宇数量对比
- 各街道建筑面积对比
- 各街道空置率排名
- 街道载体类型分布

#### 3.1.3 楼宇统计
单个楼宇的详细统计：
- 楼宇基本信息
- 楼层使用情况
- 房间状态统计
- 历史数据趋势

### 3.2 数据分析功能

#### 对比分析
- 街道间数据对比
- 时间维度对比（同比、环比）
- 载体类型对比

#### 趋势分析
- 空置率变化趋势
- 面积利用率趋势
- 新增资产趋势

### 3.3 数据导出

- 支持Excel格式导出
- 支持PDF报表导出
- 支持图表导出
- 自定义导出字段

## 4. 系统管理

### 4.1 组织机构管理

#### 功能描述
- 维护系统的组织架构
- 支持多层级组织结构
- 组织与数据权限关联

#### 数据字段
- 机构编号（GS+组织信用代码后六位）
- 机构名称
- 上级机构
- 机构类型
- 联系人
- 联系电话
- 机构地址
- 排序号
- 状态

### 4.2 用户管理

#### 功能描述
- 管理系统用户账号
- 用户与组织机构关联
- 用户与角色关联

#### 数据字段
- 用户名（唯一）
- 真实姓名
- 所属机构
- 手机号码
- 电子邮箱
- 用户角色（多选）
- 账号状态
- 最后登录时间

### 4.3 角色管理

#### 功能描述
- 创建和管理系统角色
- 配置角色的菜单权限
- 配置角色的数据权限

#### 角色类型
1. **系统管理员**：所有权限
2. **区级管理员**：全区数据权限
3. **街道管理员**：本街道数据权限
4. **普通用户**：查看权限

#### 权限配置
- 菜单权限：控制可访问的功能模块
- 数据权限：控制可查看的数据范围
- 操作权限：控制增删改查等操作

### 4.4 菜单管理

#### 功能描述
- 配置系统菜单结构
- 支持多层级菜单
- 菜单与权限关联

#### 菜单属性
- 菜单名称
- 上级菜单
- 菜单图标
- 菜单类型（目录/菜单/按钮）
- 菜单路径
- 组件路径
- 权限标识
- 排序号
- 是否显示

### 4.5 日志管理

#### 4.5.1 登录日志
记录用户登录信息：
- 登录用户
- 登录时间
- 登录IP
- 登录地点
- 浏览器信息
- 操作系统
- 登录结果

#### 4.5.2 操作日志
记录用户操作行为：
- 操作用户
- 操作时间
- 操作模块
- 操作类型（增/删/改/查/导入/导出）
- 操作对象
- 操作内容
- 操作结果
- 操作IP

### 4.6 数据字典

#### 功能描述
- 维护系统的基础数据
- 提供下拉选项的数据源
- 支持层级数据字典

#### 字典类型
- 资产类型
- 楼宇类型
- 房间类型
- 土地性质
- 载体类型
- 状态类型

### 5.1 天地图集成

#### 集成内容

- 地图数据服务

#### 功能支持

#### 接口类型
