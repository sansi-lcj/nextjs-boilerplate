## 快速导航

### 核心功能模块
1. **资产管理** - 四级资产管理体系（资产-楼宇-楼层-房间）
2. **地图展示** - 基于天地图的资产可视化展示
3. **数据统计** - 多维度数据分析与报表
4. **系统管理** - 用户、角色、权限、日志管理

### 技术栈
- 前端：React + Ant Design + TypeScript
- 后端：Golang + Gin + GORM
- 数据库：MySQL + Redis
- 地图：天地图政务版

## 文档概述

本文档集包含楼宇资产管理平台项目的需求分析和设计文档。该项目旨在建设一个楼宇资产管理系统，实现对全区楼宇资产的数字化管理。

## 项目关键信息

- **项目范围**：楼宇资产管理平台（PC端）
- **开发周期**：13周
- **主要用户**：区/街道资产管理人员
